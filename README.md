# Ping-Pong
Описание данного проекта:
Кооперативная игра Ping-Pong, созданная с использованием возможностей библиотеки Pygame.
Движение левой ракеткой реализуется с использованием клавиш W и S, а правой ракеткой с использованием стрелок клавиатуры, мяч перемещается автоматически. 
Проиграет игрок, упустивший мяч.
Использовать мой проект можно лишь с указанием моего авторства.
